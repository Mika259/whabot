# whabot
Whatsapp bot to your number

- Linux/Ubuntu...

#Installation
- install Node js (javascript)
```apt install nodejs-lts```

- install wwebjs
```npm i whatsapp-web.js
```

- install qrcode terminal
```npm i qrcode-terminal
```

- open directory
```cd whabot```

- run bot
```node bot.js```

- then scan your qr code
